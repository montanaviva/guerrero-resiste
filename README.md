# Guerrero Resiste

Proyecto ciudadano independiente de documentación y denuncia sobre desplazamiento forzado en la Montaña Baja de Guerrero, México.

## ⚠️ Aviso
- No solicitamos donaciones.
- No almacenamos datos personales.
- La vocería mantiene identidad en reserva por seguridad.
- Los datos presentados están verificados con fuentes abiertas y organizaciones de derechos humanos (Tlachinollan, CIPOG-EZ).

## Contenido
Testimonios, datos forenses, mapeo de actores responsables y acciones de presión internacional.
